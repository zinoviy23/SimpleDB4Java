Файл представляет из себя описание таблиц базы данных в виде классов языка Java.
Описание начинается с ключевого слова database, после чего следует имя базы данных.

Классы начинаются с ключевого слова class. Внутри класса могут быть не статические поля и статические методы.
В классе должно быть хотя бы одно имя. По классу генерируется таблицы в базе данных и класс языка Java с методами, описанными
пользователем и других нужных методов для работы с базой данных.
Поля могут быть типа long, int, float, Другой определенный пользователем тип и список([]).
Методы могут иметь тело {}, или быть одним выражением через ->.

Операции:
+,-,*,/,==,!=,<=, >=, <, >.
Так же есть вызов методов и взятие элемента массива.
массивы могут задаваться явно через {}.
Для всех полей генерируется get, set. Для массиво get, add, remove.
Так же есть тип boolean.
Есть цикл foreach и операто if-else.
Так же есть операции +=, -=, =. +=, -= работают только для численных типов.
+ работает так же для строк.
Строки задаются через "".

Так же есть особенные методы min/max.
Аргументы этих методов перечесление полей класса. Допускается +/-. Данные методы ищут min/max из записей в бд по переданным полям.

Метод find
принимает булево выражени. Оно может принимать только выражения с полями класса и константами.
